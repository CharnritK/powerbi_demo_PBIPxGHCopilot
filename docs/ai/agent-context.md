# Agent Context

This repo is for Power BI and analytics engineering work focused on trusted semantic-model delivery.

Fast navigation:

- reusable code: `src/`
- notebooks: `notebooks/`
- Power BI assets: `demo-enterprise/bi-repo/powerbi/workspaces/regional-sales-trust-demo/pbip/`
- sample data: `demo-enterprise/bi-repo/powerbi/workspaces/regional-sales-trust-demo/assets/data/`
- business rules and model docs: `docs/domain/` and `docs/data-model/`
- external boundaries: `docs/external-repos/`

Safe-to-edit areas:

- `src/`
- `scripts/cli/`
- `tests/`
- `config/*.example.yaml`
- most markdown docs

Edit-with-caution areas:

- `.pbip`, `.Report`, `.SemanticModel`, and `.ipynb` files
- MCP smoke-test scripts
- docs that define contracts or source-of-truth assumptions

Source-of-truth folders:

- `src/`
- `demo-enterprise/bi-repo/powerbi/workspaces/regional-sales-trust-demo/pbip/`
- `docs/data-model/`
- `docs/external-repos/`
