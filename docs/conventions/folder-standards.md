# Folder Standards

- `src/` is for reusable code only.
- `scripts/cli/` is for runnable operator entrypoints.
- `notebooks/` is for guided execution and demos.
- `powerbi/` is for PBIP, PBIR, TMDL, and local sample assets.
- `docs/` contains authoritative and reference docs.
- `references/` is non-authoritative reference material.
- `tests/` contains fixtures and unit tests only.
