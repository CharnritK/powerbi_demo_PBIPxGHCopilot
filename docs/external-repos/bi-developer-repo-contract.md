# BI Developer Repo Contract

Expected ownership in the BI Developer repo:

- report page development and report UX ownership
- shared report templates and reusable report components
- publishing and workspace handoff patterns
- downstream report acceptance and business-facing sign-off

This repo owns:

- reusable engineering helpers
- semantic-model inspection and validation utilities
- local PBIP sample and model documentation
- operator notebooks and local demo workflows

Shared expectations:

- naming conventions stay aligned across repos
- measure ownership is explicit when shared semantic-model changes affect reports
- reusable report assets are referenced through contracts, not copied blindly into this repo
