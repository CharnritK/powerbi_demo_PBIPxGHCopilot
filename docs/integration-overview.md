# Integration Overview

The demo uses a simple enterprise handoff model.

## Responsibilities

- Data Engineering publishes curated tables and views intended for analytics consumption.
- BI consumes those curated outputs rather than modeling directly from raw operational sources.
- BI owns semantic model design, measures, report logic, and business-facing presentation.

## Demo interpretation

Inside this repository, those responsibilities are represented as folders only. The integration is illustrative and documentation-led rather than technically enforced.
