# Runbooks

Common local workflows:

- Validate TMDL: `python scripts/cli/validate_tmdl_semantic_model.py`
- Export Power BI metadata: `python scripts/cli/export_metadata.py --group-id <workspace-id>`
- Generate measure docs: `python scripts/cli/generate_measure_docs.py`
- Register MCP: `python scripts/cli/setup_powerbi_modeling_mcp.py`
- Run the MCP smoke test: `python scripts/cli/run_mcp_smoke_test.py`
