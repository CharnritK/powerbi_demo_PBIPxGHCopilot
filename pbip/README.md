# PBIP Folder

This folder intentionally contains **PBIP design guidance** rather than a guaranteed-openable PBIP artifact generated by Power BI Desktop.

Why:
- A true PBIP should be created from Power BI Desktop by using **Save As > Power BI Project (.pbip)**.
- The current Microsoft documentation still labels Power BI Desktop projects as preview.
- For this showcase repo, the goal is to keep the asset teachable and source-control friendly without pretending that a hand-written PBIP folder is equivalent to a real Desktop-generated project.

See `docs/pbip_sample_design.md` for the recommended semantic model, report pages, measures, and RLS notes.
